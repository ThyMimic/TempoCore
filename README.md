TempoCore is a Java-based application meant as a precursor
to Momentum. TempoCore tracks user data through Google API
references and, employing a novel ML framework, optimizes
user workflow and schedules based on pattern recognition,
task efficiency, and time.

Upon installing TempoCore, the user can opt to have the 
application scan upwards of a month in regard to scheduling
data or other events planned on their Google calendar. 
The longer the range of time, the more effective TempoCore
will be at understanding user behavior at specific intervals
and making better judgement at task optimization.
